# Dishant Panjiyar | Portfolio

A modern, minimal portfolio website featuring a dark theme, glassmorphism effects, and smooth scroll-triggered animations.

## 🔗 Live Demo

dishantpanjiyar.com.np

## ✨ Features

- Full-screen animated hero section
- Featured project cards with hover effects
- Scrollable project list with animated underline and arrow reveal
- Glassmorphism "About" card with floating background graphic
- Minimal contact section with social links
- Fully responsive across desktop, tablet, and mobile
- Scroll-based fade-in animations

## 🛠️ Built With

- **HTML5**
- **CSS3** (custom properties, flexbox, backdrop-filter, keyframe animations)
- **JavaScript** (scroll reveal logic)
- **Google Fonts** — [Inter](https://fonts.google.com/specimen/Inter) & [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond)

## 🚀 Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/your-repo.git
   ```
2. Navigate into the project folder
   ```bash
   cd your-repo
   ```
3. Open `index.html` in your browser, or serve it locally
   ```bash
   npx serve .
   ```
